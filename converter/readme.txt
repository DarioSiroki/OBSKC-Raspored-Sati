converter je napisan u nodeJS-u i prebacen u EXE file pomocu zeit/pkg biblioteke (https://github.com/zeit/pkg)

source kod je u node source folderu

node source sadrzi i skriptu koja automatski preuzima najnoviju verziju rasporeda sa sluzbene stranice OBSKC (ne koristi se ali je ostalo za slucaj da bude potrebno ubuduce)

kako koristiti: 
smjestiti konverter (npr. convert-win.exe) u istu mapu kao i rasporede koji se zele konvertirati i samo pokrenuti exe datoteku
